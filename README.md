# Clean Architecture com Spring Boot

Aplicação simples desenvolvida com o objetivo de explorar a Clean Architecture dentro do ecossistema Spring Boot, implementando um fluxo básico de criação de conta e destacando a separação de responsabilidades entre camadas, o uso de casos de uso independentes do framework, a organização modular entre domínio e infraestrutura, além de boas práticas que favorecem a construção de serviços escaláveis e de fácil manutenção.
